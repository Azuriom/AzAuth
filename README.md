# AzAuth

A Java implementation of the Azuriom Auth API.

## Usage

You can find how to use AzAuth on our [documentation](https://azuriom.com/docs/auth-api).

## Dependencies

AzAuth use [Google Gson](https://github.com/google/gson) to deserialize Json.
